# Програмний модуль @molfar/api-manti | Вступ

Програмний модуль @molfar/api-manti – "Програмний модуль веб-сервісу для інформаційного пошуку повідомлень", який написаний мовою програмування JavaScript, призначений для інформаційного пошуку збережених повідомлень в базі даних ManticoreSearch (opens new window).

Функціональне призначення

Програмний модуль призначений для інформаційного пошуку збережених повідомлень в базі даних ManticoreSearch (opens new window).

Опис логічної структури

Програмний модуль складається з:

  • api - HTTP-сервер
  • controllers – контролер запитів
  • routers – диспетчер запитів
  • searcher - пошуковик

На HTTP-сервер api, який складається з визначеної кількості шляхів routers, приходить запит, який обробляється контролером запитів controllers. Контролер controllers за певним запитом використовує відповідні сервіси, що виконують певну логіку.

Використовувані технічні засоби

Програмний модуль експлуатується на сервері під управлінням Node.js. В основі управління всіх сервісів є система оркестрації Kubernetes, де всі контейнери працюють з використанням Docker.

Виклик та завантаження

Завантаження програмного модуля забезпечується введенням в WEB-браузері адреси завантажувального модуля http://{hostname} (opens new window) з можливими вказівками:

  • / (opens new window) метод GET для виклику сторінки із загальним описом сервісу
  • /metrics (opens new window) для виклику сторінки статистики сервісу
  • /timeline (opens new window) метод POST для отримання інформації з ManticoreSearch у вигляді графіку
  • /aggregation (opens new window) метод POST для агрегації даних з ManticoreSearch
  • /content (opens new window) метод POST для отримання інформації з ManticoreSearch
  • /dict/person (opens new window) метод GET для створення словника
